# InterFunItems
Простой плагин на предметы FunTime для PocketMine-NG

# Установка:
Загрузите файл InterFunItems в папку plugins и все готово!

# Команды:
/funitems - открытие формы с выдачей кастомных предметов
/funitems - <ник> <предмет> <количество> - выдача кастомных предметов с помощью команды

# Эффекты предметов:
Дезориентация - Замедление V на 5 сек, Слепота I на 3 сек, Тошнота I на 3 сек, Иссушение III на 10 сек, Усталость V на 5 сек. Радиус действия: 10 блоков
Снежок Заморозка - Заморозка на 1 сек, Слабость I на 5 сек. Радиус действия: 7 блоков
Огненный Смерч - Поджигает всех игроков на 3 секунды. Радиус действия: 10 блоков

# Дополнительная информация:
На все предметы установлен кулдаун в 30 секунд.
